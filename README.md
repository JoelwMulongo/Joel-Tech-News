# 📰 Joel Tech News

**Joel Tech News** is an Android application that delivers the latest and most trending technology news directly to your mobile device.  
Built with **Kotlin** and **Java**, the app provides a modern, fast, and reliable experience for tech enthusiasts who want to stay informed on the go.
